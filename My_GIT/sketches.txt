Как должно это выглядеть

./my_git.exe init Directory - должно создать дерикторию с нужными файлами

Как должна выглядеть директория

Directory/
|-- .vcs/
|   |-- HEAD (current commit) - будет хранить в себе хэш текушего коммита (символьная ссылка)
|   |-- objects/
|       |-- a1b2c3d4/ - ну видемо это хэш (айдишника)
|			|-- и тут будут лежать файлы этого коммита
|           |-- PREV - будет хранить ссылку на предыдущий коммит (символьная ссылка)
|       |-- a2b3f6a0/ 
|   |-- commits.json - хранится инфа о всех коммитах
|-- тута текущие файлы. 

Как выглядит commits.json
{
	count_commits : int
	repository_path : string
	commits: [
		{
			"id" 		: string,
			"message" 	: string, 
			"create_at" : int,
			"ttl"		: int
		}
	]
}
